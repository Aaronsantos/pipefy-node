## UNDER DEVELOPMENT (ALPHA)

An unofficial Pipefy API wrapper using node.js

## Official Reference

[Pipefy Apiary](http://docs.pipefy.apiary.io)

## Installation

```sh

    $ npm i -S pipefy-node

```

## Usage

```javascript

    var pipefy = new(require('pipefy-node'))({
        'your_personal_access_token': '<token>'
    });
    
```

## Tests

    `npm test`
